# 问题说明和解决方案

## 🔍 当前问题分析

### 问题 1：Node.js 未安装
- **现象**：运行 `npm run dev` 时提示 "找不到 npm 命令"
- **原因**：系统中没有安装 Node.js
- **影响**：无法运行 Next.js 项目

### 问题 2：Homebrew 未安装但被引用
- **现象**：每次打开终端显示 `no such file or directory: /opt/homebrew/bin/brew`
- **原因**：`.zshrc` 文件中引用了不存在的 Homebrew
- **影响**：每次启动终端都会显示警告信息（不影响功能，但很烦人）

---

## ✅ 解决步骤

### 第一步：修复 .zshrc 文件（可选，但推荐）

运行以下命令修复终端警告：

```bash
cd "/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮"
./fix_zshrc.sh
source ~/.zshrc
```

或者手动编辑 `~/.zshrc` 文件，将内容改为：

```bash
# Homebrew 配置（仅在 Homebrew 存在时加载）
if [ -f /opt/homebrew/bin/brew ]; then
    eval "$(/opt/homebrew/bin/brew shellenv)"
fi
```

---

### 第二步：安装 Node.js（必需）

**推荐方式：使用官方安装包（最简单）**

1. 访问：https://nodejs.org/zh-cn/
2. 下载 LTS 版本（推荐 v18 或 v20）
3. 双击 `.pkg` 文件安装
4. 安装完成后，**重新打开终端**，运行：
   ```bash
   node --version
   npm --version
   ```
   如果显示版本号，说明安装成功

---

### 第三步：运行项目

```bash
cd "/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮"

# 如果 node_modules 有问题，先重新安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后在浏览器中访问：**http://localhost:3000**
