# 3D立方体空间可视化

基于图片中的立方体布局创建的交互式3D可视化项目。

## 功能特点

- ✨ 3D立体空间效果
- 🖱️ 鼠标拖拽旋转视角
- 🔍 滚轮缩放
- 📦 每个立方体按图片中的命名规则标记
- 🎨 现代化的UI界面

## 使用方法

1. 在浏览器中打开 `index.html` 文件
2. 使用鼠标左键拖拽来旋转视角
3. 使用鼠标滚轮进行缩放
4. 点击"重置视角"按钮恢复默认视角
5. 点击"切换线框"按钮查看线框模式

## 立方体命名规则

根据图片中的标签系统：
- **层（Layer）**: C（前层）、B（中层）、A（后层）
- **行（Row）**: 1（顶部）、2（中间）、3（底部）
- **列（Column）**: 1（左侧）、2（中间）、3（右侧）

命名格式：`[层][行]-[列]`，例如：
- `C1-1`: 前层、顶部、左侧
- `A3-3`: 后层、底部、右侧

## 技术栈

- Three.js - 3D图形库
- OrbitControls - 相机控制器
- WebGL - 3D渲染

## 注意事项

- 需要支持WebGL的现代浏览器
- 建议使用Chrome、Firefox、Edge等最新版本浏览器
