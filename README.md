# 3D立方体空间可视化 - 照片飞轮项目

一个基于 Three.js 的交互式 3D 可视化项目，展示立方体空间布局和照片展示功能。

## 功能特点

- ✨ **3D 立体空间效果** - 使用 Three.js 构建的 3D 场景
- 📸 **照片飞出动画** - 点击 B2-2 盒子，9 张照片从盒子里飞出
- ✨ **粒子系统** - 彩色闪烁粒子效果，营造节日氛围
- 🖼️ **照片交互** - 点击照片可放大查看和对话
- 💬 **Windows 98 风格对话框** - 复古风格的聊天对话框
- 🖱️ **鼠标控制** - 拖拽旋转、滚轮缩放视角
- 🎨 **温暖灯光效果** - 圣诞和家庭氛围的灯光设置

## 快速开始

### 方法一：使用本地服务器（推荐）

由于项目使用了 ES6 模块，需要通过 HTTP 服务器运行：

```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx http-server -p 8000

# 然后在浏览器中访问
http://localhost:8000
```

### 方法二：直接打开

某些现代浏览器支持直接打开，但可能受到 CORS 限制，建议使用方法一。

## 使用说明

1. **打开盒子** - 点击屏幕上的 B2-2 立方体盒子
2. **查看照片** - 照片会从盒子里飞出，围绕盒子排列成圆形
3. **交互照片** - 
   - 鼠标悬停：照片会显示小手图标，可以左右滚动查看
   - 点击照片：照片会放大到屏幕中央，显示对话框
4. **对话框操作** - 
   - 点击"是"：所有照片和粒子收回盒子，C2-2 盒子出现
   - 点击"否"：照片平滑返回原位

## 技术栈

- **Three.js** (v0.160.0) - 3D 图形库
- **OrbitControls** - 相机控制器
- **WebGL** - 3D 渲染
- **ES6 Modules** - 模块化开发

## 立方体命名规则

根据图片中的标签系统：
- **层（Layer）**: C（前层）、B（中层）、A（后层）
- **行（Row）**: 1（顶部）、2（中间）、3（底部）
- **列（Column）**: 1（左侧）、2（中间）、3（右侧）

命名格式：`[层][行]-[列]`，例如：
- `C1-1`: 前层、顶部、左侧
- `B2-2`: 中层、中间、中间（触发盒子）
- `A3-3`: 后层、底部、右侧
