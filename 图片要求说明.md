# 盒子图片替换说明

## 你可以提供的图片格式

✅ **可以使用：固定角度的二维 PNG 图片**

## 图片要求

### 1. 文件格式
- **格式**：PNG（推荐，支持透明背景）或 JPG
- **文件名**：
  - B2-2 盒子：`B2-2-box.png` 或 `B2-2-box.jpg`
  - C2-2 盒子：`C2-2-box.png` 或 `C2-2-box.jpg`
- **保存位置**：`/public/photos/` 目录

### 2. 图片内容
- **角度**：固定角度的二维图片即可（如正面、侧面、等轴测图等）
- **背景**：建议使用透明背景（PNG），或者纯色背景
- **尺寸**：建议 512x512 到 1024x1024 像素（正方形比例最佳，但不是必须的）

### 3. 显示方式

有两种方式可以选择：

#### 方式A：平面显示（推荐，最简单）
- 图片会作为一个平面显示，始终面向相机（Billboard效果）
- 优点：简单、清晰、文件小
- 适合：二维设计、图标、Logo 等

#### 方式B：立方体贴图
- 图片会贴到立方体的6个面上
- 需要提供6个面的图片，或者1张图片重复使用
- 适合：想要保持立方体3D外观

### 4. 透明度处理
- PNG 透明背景会正确显示
- 支持 Alpha 通道

## 当前盒子尺寸

- 盒子大小：1.25 单位（在3D场景中）
- 图片会等比例缩放适配

## 实现方式

我会将当前的立方体替换为：
- 使用 `PlaneGeometry`（平面）或保持 `BoxGeometry`（立方体）
- 使用 `MeshBasicMaterial` 或 `MeshStandardMaterial`
- 加载你提供的图片作为纹理

## 推荐方案

**推荐使用方式A（平面显示）**：
- 提供一张 PNG 图片（支持透明背景）
- 尺寸：512x512 或 1024x1024 像素
- 文件名：`B2-2-box.png` 和 `C2-2-box.png`

这样实现最简单，显示效果也最清晰！
