# C2-2 图片替换说明

## 操作步骤

### 1. 准备图片文件
将你的4张图片准备好，确保文件名格式正确。

### 2. 放置图片文件
将4张图片文件放到以下目录：
```
/public/photos/
```

### 3. 文件命名规则
按照以下命名规则命名图片文件：
- 第1张图片：`C2-2-1.jpg` （或 `.JPG`, `.png`, `.jpeg`）
- 第2张图片：`C2-2-2.jpg`
- 第3张图片：`C2-2-3.jpg`
- 第4张图片：`C2-2-4.jpg`

**支持的图片格式：**
- `.jpg` 或 `.JPG`
- `.jpeg` 或 `.JPEG`
- `.png` 或 `.PNG`
- `.webp`

### 4. 示例
如果您的图片文件是：
- `图片1.jpg`
- `图片2.jpg`
- `图片3.jpg`
- `图片4.jpg`

重命名并复制到：
```
/public/photos/C2-2-1.jpg
/public/photos/C2-2-2.jpg
/public/photos/C2-2-3.jpg
/public/photos/C2-2-4.jpg
```

### 5. 注意事项
- 如果您的图片文件扩展名不同（如 `.png`），代码会自动识别，但建议统一使用 `.jpg` 或 `.JPG`
- 图片会等比例显示，不会被压缩或拉伸
- 替换图片后，刷新浏览器即可看到新图片

### 6. 文件路径
完整路径示例：
```
/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮/public/photos/C2-2-1.jpg
/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮/public/photos/C2-2-2.jpg
/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮/public/photos/C2-2-3.jpg
/Users/jiyang/Desktop/ai/备份/接ai之后/人生飞轮/public/photos/C2-2-4.jpg
```
